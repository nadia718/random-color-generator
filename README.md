# random-color-generator

A simple and interactive web application built using HTML, CSS, and JavaScript that generates random colors with a single click.

## Features

* Generate random colors instantly
* Display HEX color codes
* One-click color generation
* Dynamic background color changes
* Clean and user-friendly interface
* Responsive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

random-color-generator/
│
├── index.html
├── style.css
└── script.js

## Project Goal

This project was created while learning JavaScript to understand DOM manipulation, event handling, functions, and random value generation.

## Functionality

* Generates a random color on button click
* Displays the generated HEX color code
* Updates the page background dynamically
* Provides an interactive user experience

## Responsive Design

Optimized for:

* Desktop
* Tablet
* Mobile Devices

## Preview

Click the generate button to create a new random color and instantly update the page background.
